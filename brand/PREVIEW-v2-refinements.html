3Abstract Growth System · 5 motifs artísticos

Crescimento representado indiretamente. Sem setas, sem gráficos, sem montanhas. 5 metáforas poéticas que se tornam primitivas do sistema visual em marketing, onboarding, empty states e backgrounds editoriais.

Currents
Growth as tide
Ondas paralelas crescendo em amplitude da esquerda para a direita. Ritmo de deposits semanais.
Constellations
Growth as network
Pontos dispersos, alguns conectados por fios. Formação de capital, rede de relações.
Rings
Growth as ripple
Círculos concêntricos expandindo de origens variadas. Pedra na água — cada depósito é uma onda.
Threads
Growth as fabric
Linhas tecidas em 60°, densidade aumenta em direção. Patrimônio como fio cruzando fio.
Strata
Growth as sediment
Camadas horizontais empilhadas, cada uma ligeiramente deslocada. Tempo virando material.

Cortada da shortlist: Murmuration (bando de pássaros) — custo de SVG regenerável muito alto. Mycelium, bloom e ink-bleed também cortados. Documentado em abstract-visual-system.md. Cada família tem regras de geometria + motion spec + variantes 01/02/03 pré-aprovadas.

5Nano-banana · Experts via Gemini 2.5 Flash Image

Pacote completo para você (ou @media-engineer) executar externamente via Google AI Studio ou Gemini API. Produz os 6 experts em semi-photorealistic editorial — registro New Yorker / MIT Tech Review / Stripe Sessions. Não photorealism (uncanny valley), não illustration (perde premium).

Ready to execute

6 experts · semi-photorealistic editorial ·

Pacote contém: 1 master style constraint + 6 prompts hiper-específicos (150–250 palavras cada) + 3 caminhos de execução (AI Studio UI / Python SDK / curl) + script Python runner + recipe de post-processing + 10-point quality checklist.

~42Gerações totais (7 por expert com iteração)
$1.64Custo base (0.039/img)
<$5Custo com iterations
~60minHands-on time

Como executar (3 caminhos)

  1. Opção A — AI Studio UI (mais simples): abra aistudio.google.com → seleciona modelo Gemini 2.5 Flash Image → cola o master constraint + prompt do expert → gera 3–4 variants → escolhe melhor → repete para os 6. Total: ~45 min.
  2. Opção B — Python SDK (automatizado): pip install google-genai → export GEMINI_API_KEY=... → python nano-banana-runner.py --all --variants 4. O script já está pronto em /brand/03-ai-experts/nano-banana-runner.py. Loop automático nos 6 experts, salva em /assets/experts/real/{name}-{seed}.png.
  3. Opção C — curl (pipeline CI): exemplo completo no nano-banana-prompt-package.md §6.
  4. Post-processing: crop para 512×512 → dot overlay emerald #16A661 bottom-right via SVG composite → salva como {name}-final.png. Recipe completo no §8.
  5. Integration: <ExpertAvatar variant="png" expert="nora"/> — nova prop já especificada, default png quando arquivo existe, svg fallback (não-breaking).
