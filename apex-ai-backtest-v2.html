📡 Sinais em Tempo Real

📈 Tendência (SMA 20/50/200)
Posição do preço em relação às médias móveis determina o viés: Bullish / Neutral / Bearish. Camadas L1 bloqueadas em tendência de baixa.
〰️ Volatilidade (ATR 14)
ATR% ajusta o espaçamento entre camadas. Alta volatilidade → espaçamento maior (evitar ativação prematura). Baixa → mais tight.
💪 Força (RSI 14)
RSI > 70: sobrecomprado, reduz TP para realizar lucros rápidos. RSI < 30: sobrevende, aumenta max camadas temporariamente.
📊 Volume Profile
Volume acima da média em quedas = venda real (pior cenário). Volume baixo em quedas = liquidez esgotada (recuperação esperada).
💰 Funding Rate (Perps)
Funding positivo alto: short premium. Ajusta TP target para compensar custo de funding. Funding negativo: bônus para longs.
🎭 Regime de Mercado
Bull/Bear/Sideways detectado por combinação de sinais. Cada regime tem parâmetros específicos: TP, spacing, max layers, CB tolerance.

🔄 Pipeline de Decisão — Pseudocódigo

// A cada fechamento de candle:
signals = {
  trend:   calcTrend(close, sma20, sma50, sma200),    // bull|bear|sideways
  atrPct:  calcATR(14) / close * 100,                  // volatilidade %
  rsi:     calcRSI(14),                                // 0-100
  volume:  volume / avgVolume20,                       // ratio
  funding: getFundingRate(),                            // % por 8h
}

regime = classifyRegime(signals)                       // BULL|BEAR|SIDEWAYS|HIGH_VOL

params = REGIME_PARAMS[regime]                         // TP, spacing, maxL, CBtol

// Ajustes adicionais via RSI:
if (signals.rsi > 70) params.tp *= 0.85               // sobrecomprado → TP menor
if (signals.rsi < 30) params.maxLayers += 2            // sobrevende → mais camadas

// Ajuste de espaçamento:
spacing = clamp(signals.atrPct * params.atrMult, 0.5, 5.0)

// Verificar deploy de reserva:
if (cycle.drawdown > params.cbTol * 0.8) deployReserve('protection')
if (cycle.days > 45 && cycle.layers >= maxLayers)  deployReserve('strategic_close')

// Decisão de abrir L1:
canOpenL1 = regime != 'BEAR' && !cbBlocked && signal.trend != 'strong_down'
